核心技术：DeepSeek API + Prompt Engineering

DeepSeek 大模型特性

128K token 上下文窗口，可处理完整论文草稿

强大的中文理解和结构化信息提取能力

通过 API 调用，无需本地部署

Prompt Engineering 策略

专用 System Prompt 引导输出规范 JSON

明确定义标题/作者/摘要/章节/参考文献识别规则

temperature=0.3 确保一致性和可复现性
